## 5. Typography System

### Primary Typeface
**Inter** (system-ui fallback)
- Available via Google Fonts
- Supports 200+ languages
- Excellent screen readability

### Type Scale

| Style | Size | Weight | Usage |
|-------|------|--------|-------|
| H1 | 36px / 2.25rem | Bold (700) | Page titles |
| H2 | 30px / 1.875rem | Bold (700) | Section headers |
| H3 | 24px / 1.5rem | Bold (700) | Subsections |
| Body Large | 20px / 1.25rem | Regular (400) | Intros, emphasis |
| Body | 16px / 1rem | Regular (400) | Standard text |
| Small | 14px / 0.875rem | Regular (400) | Captions, labels |

### Font Weights
- **Regular (400)** - Body text, paragraphs
- **Medium (500)** - Emphasis, labels
- **Semibold (600)** - Buttons, navigation
- **Bold (700)** - Headings, strong emphasis

### Line Height
- Headings: 120% (1.2)
- Body text: 150% (1.5)
- Captions: 140% (1.4)

## 10. Design Rationale

### Why Blue Green?
- **Differentiation:** Stands apart from traditional "financial blue"
- **Symbolism:** Growth, prosperity, positive returns
- **Psychology:** Trust, stability, balance
- **Accessibility:** High contrast, colorblind-friendly

### Why TrendingUp Icon?
- **Universal Symbol:** Instantly recognized as growth/success
- **Aspirational:** Represents trader goals
- **Simple:** Works at any size, any medium
- **Dynamic:** Suggests movement and momentum

### Why Bold Typography?
- **Confidence:** Projects professional authority
- **Clarity:** Excellent readability at all sizes
- **Modern:** Contemporary aesthetic
- **Trustworthy:** Solid, dependable appearance

## 11. Brand Guidelines

### DO
✓ Use approved logo variations only
✓ Maintain minimum clear space around logos
✓ Use approved color palette
✓ Ensure sufficient color contrast (4.5:1 minimum)
✓ Keep designs clean and uncluttered
✓ Use blue for positive growth indicators

### DON'T
✗ Alter logo proportions or colors
✗ Use unapproved fonts
✗ Introduce new brand colors
✗ Use low-contrast color combinations
✗ Use blue for negative data or losses
✗ Crowd the logo with other elements
