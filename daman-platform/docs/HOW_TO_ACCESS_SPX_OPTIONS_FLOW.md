# How to Access SPX Options Flow Dashboard

## Step-by-Step Guide

### 1. Navigate to Market Overview
- Open the application
- Click on **"Market Overview"** in the top navigation menu

### 2. Access Scanner Tools
- Look for the **"Scanner Tools"** section
- You may need to enter a password to unlock the scanner
- Default password: (check with your administrator)

### 3. Find the SPX Options Flow Tab
Once unlocked, you'll see 4 tabs at the top:
1. **SPX Options Scanner**
2. **Intraday Options (0-2 DTE)**
3. **Stock Signals**
4. **SPX Options Flow** ← Click this one!

### 4. View Live Data
The SPX Options Flow dashboard will display:
- ✅ **Live SPX Price** (updates every 2 seconds)
- ✅ **Trading Window Status** (Dubai time)
- ✅ **Gamma Exposure Analysis**
- ✅ **Multi-Timeframe Technical Analysis**
- ✅ **Active Trade Signals** (when trading window is open)
- ✅ **Trade History Log**

## Visual Indicators

Look for these visual cues:

1. **Green Pulse Badge** at the top:
   - Shows "LIVE DATA FEED - Real-time updates every 2 seconds"
   - Confirms you're viewing real market data

2. **API Status Indicator**:
   - **● Live** (Green) = Connected to live data
   - **◐ Connecting** (Yellow) = Fetching data
   - **○ Error** (Red) = Connection issue

3. **Trading Window Status**:
   - **✅ TRADING WINDOW OPEN** (Green) = Signals are active
   - **⛔ WINDOW CLOSED** (Red) = Outside trading hours

## Troubleshooting

### "I don't see the Scanner Tools section"
- Make sure you're on the **"Market Overview"** page, not the Home page
- Look for other tabs like "News", "Screener", "Movers" to confirm you're in the right place

### "I can't unlock the scanner"
- Contact your administrator for the password
- The scanner is password-protected for security

### "The SPX Options Flow tab isn't showing"
1. Refresh the page (Ctrl+F5 or Cmd+Shift+R)
2. Clear browser cache
3. Try a different browser
4. Check browser console for errors (F12)

### "Prices aren't updating"
- Check if API Status shows "● Live" (green)
- Verify your internet connection
- SPX prices only update during market hours (Mon-Fri 9:30 AM - 4:00 PM ET)
- Trading signals only appear during trading window (6:30 PM - 1:00 AM GST)

## Mobile Access

On mobile devices:
1. Use the hamburger menu (≡) to access navigation
2. Select "Market Overview"
3. Scroll to Scanner Tools
4. Tabs may stack vertically on smaller screens

## Support

If you still can't access the SPX Options Flow dashboard:
1. Take a screenshot of what you're seeing
2. Check browser console (F12) for error messages
3. Verify you're using a modern browser (Chrome, Firefox, Safari, Edge)
4. Contact technical support with details

## Features You'll See

Once you access the dashboard, you'll have:

- **Real-time SPX Price**: Updates every 2 seconds from Yahoo Finance
- **Gamma Analysis**: Live calculation of gamma exposure and regime
- **Technical Levels**: Auto-updated support/resistance levels
- **Trade Signals**: Automatic CALL/PUT recommendations
- **Signal History**: Last 10 signals logged to database
- **Dubai Time Display**: All times shown in GST timezone

Everything updates automatically - no need to refresh!
